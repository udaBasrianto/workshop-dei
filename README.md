# DeiStok - Point of Sale (POS)

Sistem Kasir (POS) modern yang dibangun dengan arsitektur modern: **Go** untuk Backend dan **React** untuk Frontend.

## Struktur Project

- **/backend**: API Server menggunakan bahasa pemrograman Go.
- **/frontend**: Aplikasi Client menggunakan React + Vite + TailwindCSS.

## Cara Menjalankan

### Backend
1. Masuk ke folder `backend`
2. Jalankan `go run main.go`

### Frontend
1. Masuk ke folder `frontend`
2. Jalankan `npm run dev`

## Fitur Utama
- Dashboard Statistik Real-time
- Point of Sale (Kasir) Responsif
- Manajemen Produk & Stok
- Manajemen Bahan Baku & Tenaga Kerja (HPP)
- Laporan Keuangan Otomatis
- Sistem Notifikasi Premium
